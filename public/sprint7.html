04/02 JSON En dat een JSON object uit een lijst (array) van meerdere objecten kan bestaan: [ { "name": "FDND Student", "id": 234, "github_handle": "fdnd" }, { "name": "Iemand anders", "id": 456, "github_handle": null } ] JSON is een standaard die veel gebruikt wordt in JavaScript. Niet geheel ontoevallig, want JSON staat voor JavaScript Object Notation. Alles heeft een haakje, behalve null en cijfers. Vergeet de kommas niet, die horen bij elke value behalve de laatste. Alles wat aan de linkerkant komt is de property en de rechterkant is de value. Data ophalen in debrowser heet een fetch Document
